- Hero section with headline, description, and placeholder image

- Testimonial quote section  
- Call-to-action banner  
- Footer with copyright text  
- Consistent color palette and typography from the design spec

---

# **Key Learnings**

- Building a page step-by-step using semantic HTML  
- Using Flexbox for horizontal and vertical layout patterns  
- Controlling spacing with padding, margin, and container widths  
- Avoiding common layout issues related to default browser styles  
- Matching a visual reference by analyzing structure before coding  
- Writing organized and reusable CSS
